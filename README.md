# configs

Opinionable configs

## Usage

```shell
$ pnpm add @ntnyq/eslint-config @ntnyq/prettier-config -D
```

```json5
// package.json
{
  "prettier": "@ntnyq/prettier-config",
  "eslintConfig": {
    "root": true,
    "extends": "@ntnyq/eslint-config"
  }
}
```
