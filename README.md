## Renovate config

Config in `.github/renovate.json`:

```json
{
  "$schema": "https://docs.renovatebot.com/renovate-schema.json",
  "extends": ["github>ntnyq/configs//packages/renovate-config/default.json"],
  "automerge": true
}
```
