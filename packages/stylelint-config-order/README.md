# stylelint-config-order

> ntnyq's stylelint config with order

## Usage

### Install

```bash
$ npm i @ntnyq/stylelint-config-order -D
# OR
$ yarn add @ntnyq/stylelint-config-order -D
```

### Extend this config

in `.stylelintrc.yml`

```yml
extends:
    - '@ntnyq/stylelint-config-order'

rules:
    # Override rules
```

in `package.json`

```json
{
    "stylelint": {
        "extends": "@ntnyq/stylelint-config-order"
    }
}
```
