Opinionable and shareable tsconfig of [ntnyq](https://ntnyq.com).

## Install

```shell
npm install @ntnyq/tsconfig -D
```

```shell
yarn add @ntnyq/tsconfig -D
```

```shell
pnpm add @ntnyq/tsconfig -D
```

## Usage

Preset `base`:

```json
{
  "extends": "@ntnyq/tsconfig/base.json",
  "compilerOptions": {}
}
```

Preset `strict`:

```json
{
  "extends": "@ntnyq/tsconfig/strict.json",
  "compilerOptions": {}
}
```
