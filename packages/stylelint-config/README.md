# StyleLint config

## Usage

### Install

```bash
$ npm i @ntnyq/stylelint-config -D
# OR
$ yarn add @ntnyq/stylelint-config -D
```

### Extend this config

in `.stylelintrc.yml`

```yaml
extends:
  - @ntnyq/stylelint-config
```

in `package.json`

```json
{
    "stylelint": {
        "extends": ["@ntnyq/stylelint-config"]
    }
}
```
